# Uso

## Casos de uso

Este proyecto está diseñado para ser un ejemplo de documentación profesional.  
Algunos casos de uso incluyen:

1. Estudiantes que quieren aprender Markdown y GitHub Pages.  
2. Equipos de desarrollo que necesitan un ejemplo de documentación clara.  

---

## Pasos de uso

1. Abrir la documentación desde GitHub Pages:  
[Documentación del proyecto](https://hasnae-eddakri.github.io/proyecto-intermodular-doc/)

2. Navegar por las secciones usando el índice o los enlaces.  

3. Consultar los archivos `.md` en local si deseas editar o ampliar la documentación.
